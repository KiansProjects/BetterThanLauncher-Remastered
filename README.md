# BetterThanLauncher

**BetterThanLauncher (BTA Launcher)** is a custom launcher for **[BetterThanAdventure](https://www.betterthanadventure.net/)**.  
It allows you to manage game instances, modify settings, and run the latest versions of BTA with ease.

## Installation

1. Download the latest release from [Releases](https://github.com/KiansProjects/BetterThanLauncher-Remastered/releases)
2. Extract the `tar.gz` file to a folder of your choice
3. Run `BetterThanLauncher.exe`
4. Start creating and managing your game instances!
